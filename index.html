    Welcome to GitHub Pages.

    This is my custom tutoiral on how to get started with GitHub Pages and how to lunch your own site! Maybe with your own domain as well. What is GitHub Pages? Pages is a easy and simple way to be able to post your project to the internet. My website Zak Humphrey is completly hosted off GitHub Pages!

    Step One: Getting Started With Pages

    The first step to makeing a github pages site, is simple, really.

    Next you click on: New Repository

    Then you name the repository anything that you would like.

    Now you click on Create Repository

    Then click on Settings

    Once you're in Settings Scroll down a bit and click on Launch automatic page generator

    And from there you can customize this page however you like.

    Once done customizing however you please, click on Continue to Layouts

    Once you chosen your layout, click on Publish page

    Tada! You're Finnished!

    Now to check what the link of your page is, just go to settings, then scroll down to github pages, it should look something like: "user.github.io/NameOfProject"
